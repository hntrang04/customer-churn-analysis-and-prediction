# data-analysis-and-visualization
Dự án phân tích khám phá dữ liệu và trực quan hóa về hành vi rời bỏ khách hàng (Churn) trong thương mại điện tử. Dự án tập trung vào tiền xử lý dữ liệu, phân tích khám phá, kiểm định thống kê và thiết kế trực quan nhằm phát hiện các mẫu hình và mối quan hệ liên quan đến churn.
